# 정간보 생성기 — 기능·설정 총정리

정간보(井間譜) 생성기 웹앱이 제공하는 거의 모든 기능과 설정을 정리한 사용자용 문서입니다.

## 0. 개요

- **구조**: `index.html`(마크업) + `app.js`(전체 로직) + `styles.css`(스타일) + 데이터 파일 3개(`notes-data.js` 율명·기호 이미지, `symbols-data.js` 시김새 79종 SVG, `janggu-data.js` 장구 구음 7종). 이미지가 전부 base64로 내장되어 있어 외부 리소스 없이 동작합니다.
- **서버/DB 없음**: 정적 파일만으로 동작하는 클라이언트 사이드 앱입니다.
- **자동 저장**: 브라우저 `localStorage`에 실시간 자동 저장되어 새로고침해도 유지됩니다(11장 참고).
- **읽는 순서**: 전통 방식 그대로 **오른쪽 → 왼쪽**으로 각(세로줄)이 진행됩니다.

### 화면 구성

| 영역 | 내용 |
|---|---|
| 상단바 | 새 문서(＋) · 실행 취소(↶) · 다시 실행(↷) · 전체 초기화 · **입력 방식**(에디터/직접 입력) 전환 + 안내(?) · 오른쪽에 **재생 컨트롤**(▶/■/⚙)·**보기 배율**(−/＋/맞춤)·**다크/라이트**(아이콘+글자) 전환 고정 |
| 오른쪽 사이드바 | **문서 / 레이아웃 / 출력** 3개 탭(접을 수 있음, "설정" 버튼으로 다시 열기) |
| 가운데 악보 영역 | SVG 악보 미리보기 (줌 바·재생 바는 상단바로 옮겨져 항상 같은 자리에 고정) |
| 기능바(리본) | 각 삽입/삭제 · 셀 서식 · 율명 크기 등. 에디터 모드에선 아래 편집 독 맨 위, 직접 입력 모드에선 **기본으로 왼쪽 세로 사이드에 도킹**(위치 전환 버튼으로 악보 위 가로 배치와 오감, 도크 폭은 손잡이를 끌어 조절 — 14장 참고). 좁으면 줄바꿈으로 접혀 항상 전부 보임. 접기(∨) 가능 |
| 아래 편집 독(에디터 모드) | 왼쪽 레일 탭(선율/장단/가사/텍스트/셀 서식)으로 편집기를 하나씩 전환. 위 경계를 끌어 높이 조절 |
| 뜬 도구창(직접 입력 모드) | 율명·시김새·장단·가사·텍스트·**각/장**·셀 서식 창이 한 번에 하나만 열림. 기본(왼쪽 도킹)에선 기능바 아래에 도킹되어 사이드바처럼 정리되고, 가로 배치에선 악보 위에 떠서 그립(⠿)으로 끌어 이동. 오른쪽 위 ✕로 닫기 |

## 1. 새 문서 만들기 (마법사)

상단바의 **＋(새 문서)** 버튼을 누르거나, **저장된 작업이 전혀 없는 첫 방문**이면 자동으로 마법사가 뜹니다.

| 항목 | 설명 |
|---|---|
| 한 각의 정간 수 | 한 각(세로줄)에 들어갈 정간(칸) 수. 기본 12 |
| 대강 분절 (선택) | 굵은 선으로 나눌 정간 묶음(예: `3 3 3 3`). 비우면 박 수에 맞는 대표 패턴 자동 입력 |
| 총 각 수 | **비워두면 기본 20각**(회색 안내 표시). 만든 뒤에는 이 값이 그대로 유지되고 자동으로 늘어나지 않음 |
| 곡 제목 (선택) | 비우면 제목 없이 생성. 옆에서 **세로 칸 / 가로 위** 표기 방향 선택 |
| 부제 (선택) | 제목 아래 작은 글씨. 제목·부제 모두 `//`를 넣으면 그 자리에서 줄바꿈 |
| 장단 입력 사용 | 켜면 장단 줄이 곡 맨 앞에 추가됨 |

새 문서를 만들기 전에 **지금 작업을 임시저장할지** 물어봅니다. 전체 초기화(상단바 ⟲)는 이와 달리 리로드 없이 **내용만**(선율·장단·가사·텍스트·셀 서식) 지우고 레이아웃 설정은 유지하며, ⌘/Ctrl+Z로 복구할 수 있습니다.

## 2. 입력 방식 2가지

상단바에서 언제든 전환할 수 있고(직접 입력이 앞·기본), 선율·장단·가사 모두에 공통 적용됩니다.

### 직접 입력 모드 (기본)
- **악보의 정간을 클릭하면 그 자리에 입력 카드**가 열려 보이는 그대로 입력합니다. 문법을 몰라도 됩니다.
  - `Enter`/`Tab` 다음 정간, 방향키로 인접 정간 이동, `Esc` 닫기.
  - **처음 정간을 클릭하는 순간 율명 팔레트가 한 번 자동으로 열립니다** — 닫으면(✕) 그 뒤로는 다시 저절로 열리지 않고, 시김새 등 다른 창을 열어뒀으면 그대로 둡니다.
- **기능바는 기본으로 왼쪽 세로 사이드에 도킹**되어 있습니다(위치 전환 버튼으로 악보 위 가로 배치와 오갈 수 있음 — 14장 참고).
- 기능바의 **입력** 그룹(율명·시김새·장단·가사·텍스트·각/장 버튼)으로 뜬 도구창을 열고 닫습니다(한 번에 하나). 도구창은 그 버튼을 다시 누르거나, **창 오른쪽 위 ✕(닫기)** 버튼으로 닫을 수 있습니다.
- 팔레트를 클릭하면 지금 열려 있는 칸에 바로 들어갑니다.
- **시김새 숫자 단축키**: 붙임표 시김새 칩에 숫자(1~0) 배지가 항상 붙어 있습니다. **숫자키를 누르고 있는 동안** 악보의 음을 클릭하면 그 옆에 바로 붙고, 키를 떼면 해제됩니다. 여러 음에 연달아 붙일 땐 **칩을 클릭**하면 고정(다시 누르면 해제되는 켜기/끄기)됩니다. 배정은 시김새 창의 **단축키 변환**에서 바꿉니다.

### 에디터 모드
- 곡 전체를 텍스트로 쭉 보며 편집. 아래 편집 독의 레일 탭(선율/장단/가사/텍스트/셀 서식)으로 전환합니다.
- 반복 구절 복사·붙여넣기, 문법을 이용한 대량 수정에 강합니다.
- 악보의 정간을 클릭하면 텍스트 에디터의 해당 칸으로 커서가 이동(양방향 동기화, 현재 칸은 악보에 노랗게 하이라이트).

## 3. 선율 입력

### 텍스트 문법 (에디터·텍스트 입력 공통)

| 문법 | 의미 |
|---|---|
| 줄바꿈(⏎) | **각**(세로줄, 마디) 구분 |
| `\|` | **정간**(칸, 1박) 구분 |
| 정간 안 공백(스페이스) | **분박** — 그 칸 안에서 위→아래로 줄이 나뉨(2·3분박 등 박 등분) |
| 공백 없이 이어 쓰기 | **붙임** — 한 분박(행) 안에서 왼→오른쪽 가로 배치 |
| 한글 율명 | 황·대·태·협·고·중·유·임·이·남·무·응 (12율) |
| 옥타브 접두어 | 청(+1) · 중청/겹청(+2) · 배/탁(−1) · 배탁/하배(−2). 없으면 중성 |
| `-` | 이음(연음) — 앞 음을 지속 |
| `쉼` / `쉼표` | 쉼표(무음) |
| 음 뒤 `<` | 숨표 — 그 정간의 오른쪽-아래 모서리에 고정 표시 |
| `{이름}` `[이름]` `(이름)` | 시김새 삽입 — 세 괄호 모두 동일하게 동작. 예: `황{미는표}` |
| `{이름@크기,좌우,상하}` | 시김새 개별 크기·위치 조정(4장 참고) |

- 격자는 입력된 `|` 개수가 아니라 **"한 각의 정간 수" 설정을 기준**으로 항상 온전히 그려집니다(빈 칸은 빈 채로 유지).
- 예: `황 태|협<|임` = 정간 3개 — 1번째는 황·태 분박, 2번째는 협+숨표.

### 텍스트 에디터 (선율)
- **그리드 정렬**: 각 정간 뒤에 탭이 자동 삽입되어 줄이 달라도 `|`가 세로로 나란히 정렬됩니다.
- **문법 검사**: 해석되지 않아 악보에 못 옮기는 글자는 빨간 표시로 알려줍니다.
- **페이지 조각**: 곡이 여러 페이지면 에디터에 현재 페이지 분량만 보여주고 ◀ 1/N ▶ 페이지 넘김이 나타납니다.
- 에디터 글자 크기 −/＋ 버튼, `Enter` = 커서 위치에서 각 나누기, `⌘/Ctrl+Enter` = 각 추가.

### 율명 팔레트

| 설정 | 내용 |
|---|---|
| 표 / 건반 | **표**: 음역(하배~중청) × 12율 매트릭스. **건반**: 피아노 건반 위 율명(클릭 입력 + 미리듣기) |
| 조 | 12율 전체 / 황종 평조 / 황종 계면조 / 중려 평조 — 그 조의 구성음만 모아 표시 |
| 표기 | 한자(黃·太…) / 한글(황·태…) — 팔레트와 악보 렌더에 동일 적용 |
| 황 음고 | E♭4 / C4 — 팔레트 클릭 소리·건반·악보 재생에 함께 적용 |
| 입력 토글 | 끄면 클릭해도 입력되지 않고 소리만 들려줌 |
| 소리 토글 | 클릭 시 그 음을 짧게 들려줄지 |
| 크기 −/＋ | 팔레트 표시 크기(시김새 팔레트와 따로 조절) |

### 율명 표기 규칙
- **한자 모드**(기본): 옥타브가 있으면 변형 한자(청황=潢 등)를 우선 쓰고, 변형자가 없는 조합(하배이·중청대)만 기본 한자 + 옥타브 점(●)으로 옅은 회색 표시. ±2옥타브까지 지원.
- **한글 모드**: 접두어+한글 그대로. 2글자 68%, 3글자 50%로 자동 축소되어 칸을 넘지 않습니다.
- 악보의 율명 글자만 기본보다 1.15배 크게 그려지고, **율명 크기** 슬라이더(기능바, 0.7~2.5×)로 전체 조절— 시김새 크기는 이 배율을 타지 않습니다.

### 이음(-) 전용 메트릭 (전통 관행)
- 분박 여러 줄 중 **한 줄에 `-`만 홀로** 있으면 그 줄의 세로 비중을 0.68로 줄여 낮게 누르고, 남는 공간을 음표 줄들이 나눠 가져 가운데로 모입니다. `-` 글자도 0.85배로 살짝 작게.
- `-`는 어디서든 가로로 1.95배 늘여 그려 짧아 보이지 않게 하고, 글리프가 낮게 찍히는 것을 감안해 살짝 위로 올려 세로 중심을 맞춥니다.
- 정간에 음이 정확히 둘일 때(세로 분박이든 가로 붙임이든)는 간격을 20% 좁혀 헐렁해 보이지 않게 합니다.

## 4. 시김새 (79종)

- 구성: **붙임표 59종**(음표 오른쪽에 작게 붙음 — 미는표·흘림표·농음표 등, **이름 미정 추가 기호 s00~s25 20종 포함**) + **독립 기호 18종**(한 칸을 통째로 차지 — 전성·자출 등) + **퇴성·추성 2종**(둘 다 가능).
  - ※ `s00`~`s25`는 새로 추가된 시김새로, 정식 이름이 정해지기 전까지 임시로 s번호로 표시합니다(토큰도 `{s11}` 꼴). 이름이 정해지면 표시 이름만 바뀝니다.
- 넣는 법 3가지:
  1. **팔레트 클릭** — 에디터: 커서 자리에 / 직접 입력: 지금 열린 칸에.
  2. **글자로 입력** — 음 뒤에 `{미는표}`·`[미는표]`·`(미는표)` (세 괄호 동일).
  3. **숫자 단축키**(직접 입력 전용) — 2장 참고.
- **크기/위치 미세조정 모드**(⌖ 버튼 — 직접 입력의 시김새 창, 에디터 모드의 팔레트 시김새 보기 양쪽에 있음): 켜면 악보의 조절 가능한 시김새에 옅은 네모가 표시되고, 클릭하면 조정 패널이 떠서 **크기(40~300%) / 좌우 / 상하(±90%)** 를 버튼으로 조절합니다. "위치·크기 초기화"와 "삭제"(Backspace/Delete 키로도) 제공. 팔레트 클릭·탭 전환 같은 **다른 조작을 하면 저절로 꺼져서** 켜둔 걸 깜빡하고 엉뚱한 곳을 만질 일이 없습니다.
- 조정값은 선율 텍스트에 `{이름@크기,좌우,상하}`로 저장됩니다(예: `황{미는표@120,5,-5}` = 120% 크기, 오른쪽 5·위 5 이동). 기본값(100,0,0)이면 이름만 남습니다.
- 시김새는 "율명 크기" 슬라이더의 영향을 받지 않아 음표만 키워도 시김새는 그대로입니다.

## 5. 장단(장구) 입력

- 장단 창(또는 탭) 머리의 체크박스를 켜면 활성화됩니다(새 문서 마법사에서도 선택 가능).
- 팔레트 7종: **덩 · 작은덩 · 기덕 · 궁 · 덕 · 다 · 더러러러**. 문법은 선율과 같이 `|`로 정간 구분, 공백으로 분박.
- 장단은 반복 패턴이므로 **곡 맨 처음 각(가장 오른쪽) 옆에 한 번만** 표시되고, 일반 각과 같은 폭을 차지합니다(첫 각이 그만큼 왼쪽으로 밀림). 정간 수는 선율과 자동으로 맞춰집니다.
- 장단 각은 **네 변을 일반 각보다 조금 굵은 선**으로 둘러 율명 각과 한눈에 구분되고, 각 아래(각 번호 자리)에 옅은 회색 **'장단' 라벨**이 붙습니다 — 각 번호 표시 설정과 한 세트로 켜지고 꺼집니다.
- '다'는 원본 기호가 작은 점이라 다른 구음의 0.3배 크기로 그려집니다.
- **장단 입력** 줄 오른쪽 끝의 **초기화** 버튼으로 장단만 비울 수 있고, ⌘/Ctrl+Z로 복구합니다.

## 6. 가사 입력

- 가사 창(또는 탭) 머리의 체크박스를 켜면 활성화됩니다.
- 선율과 같은 구조 — 줄바꿈 = 각, `|` = 정간, 정간마다 한 글자(또는 여러 자)를 적으면 **그 정간의 오른쪽에 딱 붙은 가사 칸**에 표시됩니다. 가사를 켜도 **각 사이 간격은 변하지 않습니다** — 가사 칸이 원래 간격 안에 들어갑니다.
- **글씨체** 선택(가사 입력 줄 오른쪽): 궁서체(붓글씨)·고딕체·명조체·애플고딕·해서체.
- **기호 팔레트**(가사 창의 "기호" 줄): 클릭하면 편집 중인 가사 칸(또는 에디터 커서)에 `{기호}` 토큰이 한 글자처럼 들어가고, 악보엔 이미지로 그려집니다.
  - 가로막대·세로막대·늘임표·뜰·모지·장지·튕김·연튕김 8종 + **전성·퇴성·추성**(시김새와 같은 그림) 3종.
  - 한 분박에 토큰이 여럿(`{모지}{퇴성}`)이면 **옆이 아니라 위아래로** 살짝 붙여 쌓이고, 각각의 크기는 단독일 때와 같습니다.
  - 기호 크기는 글자처럼 **분박 수와 무관하게 고정** — `{튕김} - {전성}`처럼 행이 늘어도 줄지 않습니다.
- 각 수·정간 수는 선율과 1:1 — 각을 추가/삭제하면 가사도 같이 밀리고 당겨집니다. 타이핑 중 선율과 각 수가 어긋나면 경고 배너가 떠서 알려줍니다(장단도 정간 수 어긋남 경고 동일).
- 여러 페이지면 선율 에디터처럼 페이지 넘김이 나타납니다. **가사 입력** 줄 오른쪽 끝의 **초기화** 버튼(⌘Z 복구)도 동일.

## 7. 텍스트 (자유 주석 — 대여음 등)

- 텍스트 창(또는 탭)에서 내용을 적고 **추가**를 누르면 악보 위에 얹힙니다.
- **악보에서 직접 끌어** 위치를 옮기고, 클릭하면 작은 패널이 떠서 크기 ±와 삭제를 할 수 있습니다.
- 목록에서 항목별로 **내용 · 방향(세로/가로) · 크기(2~30mm) · 자간(−3~20) · 글자 색 · 삭제**를 조절합니다.
- 글꼴은 "제목 글씨체" 설정을 따릅니다.

## 8. 각/장 (각 위 라벨 — 1장·대여음 등)

특정 각(마디) **위**에 붙는 이름표입니다. 자유 텍스트와 달리 **각에 소속**되어 각을 따라다니고(각 삽입/삭제·페이지 이동 포함), 인쇄·PNG에도 표시되는 악보 내용입니다.

- **넣는 법 2가지**:
  1. 악보에서 **각 위의 빈 부분을 클릭** → 그 자리에 입력 카드 (`Enter` 확정, `Esc` 취소, 비우고 확정하면 삭제).
  2. 각/장 창(입력 그룹 **章 각/장** 버튼 / 에디터 모드 레일 탭)에서 각 번호와 이름을 적고 **추가**. 창의 목록에서 수정·삭제도 합니다.
- **한자 자동 표기**: 입력은 한글 그대로 저장되고 악보 표기만 바뀝니다.
  - `1장`·`2장`처럼 쓰면 → **一章·二章** 꼴 한자로.
  - **대여음·중여음·여음·환입·초장·종장**도 한자(大餘音 등)로. 그 밖의 단어는 쓴 그대로.
- **일괄 크기·간격**(창 머리줄): 크기(×)는 모든 각/장 이름 글자에 한 배율로 — 기준이 '맨 아래 글자'라 크기를 키워도 **위로만** 자랍니다. 간격(mm)은 각 위쪽 선과 맨 아래 글자 사이 거리.
- 위 공간이 모자라면(긴 이름·페이지 위 여백) 잘리지 않게 글자가 자동으로 줄어듭니다.
- **템포 체크**(창 머리줄): 켜면 맨 처음 각(장단이 있으면 장단 칸) 위에 `一分・六十井` 식 템포 표기 — 숫자는 재생 설정(⚙)의 BPM을 따르고, 크기·간격은 각/장 설정과 같습니다.
- 각 번호는 사이드바의 **각 번호 표시**(기본: 화면에만)를 켜 두면 바로 확인할 수 있어 짝을 이룹니다.

### 문서 탭

| 항목 | 내용 |

| 종이 방향 | 세로(Portrait) / 가로(Landscape), A4 기준 |

크기 설정이 A4 인쇄영역을 넘으면 비율을 유지한 채 자동 축소되고, 출력 탭 요약에 축소율이 표시됩니다.

### 출력 탭

- **요약(readout)**: 그린 각 수·페이지 수·가로 각/세로 밴드·실제 각 너비와 간격(mm)·A4 맞춤 축소율·읽는 순서를 항상 보여줍니다.
- **인쇄 / PDF 저장** — 브라우저 인쇄. 화면 전용 UI는 모두 숨겨지고 악보만 페이지별로 출력. **PDF로 저장하면 기본 파일명이 곡 제목**이 됩니다(⌘/Ctrl+P로 인쇄해도 동일).
- **PNG 다운로드** — 페이지별 SVG를 300dpi 기준으로 렌더해 저장. 여러 장이면 `제목-1.png, 제목-2.png…`로 나뉨.
- **파일로 저장 / 불러오기** — 문서 전체(설정+선율+장단+가사+텍스트+셀 서식)를 `.jgb.json` 파일로 내보내고 다시 불러옵니다. **다른 컴퓨터로 옮기거나 남에게 보낼 때는 이 방법**을 씁니다.
- **임시 저장 (이 컴퓨터)** — 이름을 적고(비우면 제목·시간) 저장하면 목록에 쌓이고 언제든 되불러옵니다(불러오기는 ⌘Z로 취소 가능). 이 컴퓨터 브라우저 안에만 남는 저장입니다.

## 12. 재생 (사인파)

- 상단바 오른쪽의 재생 컨트롤: **▶ 재생** 버튼 하나가 상태에 따라 **⏸ 일시정지 → ▶ 이어하기**로 바뀌고, 옆에 **■ 정지**와 ⚙ 설정이 있습니다.
- 설정: **황 음고**(E♭4 기본 / C4 — 팔레트의 황 음고와 연동) · **템포**(BPM 20~240, 기본 60). 악보에 템포를 표기할지는 **각/장 창의 '템포' 체크**에서 정합니다(8장 참고).
- 템포 표기를 켜면 맨 처음 각(장단이 있으면 장단 칸) 위에 `一分・六十井`(1분에 60정간) 식으로 한자 세로 표기 — 크기·간격은 각/장 라벨 설정을 따릅니다.
- 재생 규칙: 정간 1칸 = 1박, 분박은 등분, **빈 정간·이음(-)은 앞 음 지속**, **쉼표만 무음**. 시김새는 소리에 반영되지 않음.
- 재생 중 현재 정간이 악보에 하이라이트되고 끝나면 자동 정지.

## 13. 되돌리기

- **전역 하나로 통일**: `⌘/Ctrl+Z` 실행 취소, `⌘/Ctrl+⇧+Z` 다시 실행(상단바 ↶ ↷ 버튼과 동일).
- 문서 내용 변경만 단계로 남습니다 — 보고 있는 탭, 입력 방식, 열어둔 도구창 같은 화면 상태는 되돌리기에 끼지 않습니다.
- 장단/가사 초기화, 전체 초기화, 임시 저장 불러오기도 모두 이 되돌리기로 복구합니다.

## 14. 화면 보기·조작

- **마우스 모드**(기능바 맨 왼쪽): **선택**(화살표)은 정간을 클릭해 입력·편집하는 기본 모드, **이동**(손)은 악보를 잡고 끌어 화면을 움직이는 팬(pan) 모드입니다. 이동 모드에선 커서가 손 모양으로 바뀌고 실수로 편집되지 않게 악보 클릭이 잠깁니다(확대해 놓고 이리저리 볼 때 편함). 새로고침하면 선택 모드로 돌아옵니다.
- **보기 배율(줌)**: 상단바 오른쪽에 고정 — −/＋, **세로맞춤**(페이지 높이를 화면에), **가로맞춤**(페이지 너비를 화면에). 화면 표시만 바뀌고 출력 크기와는 무관.
- 뜬 도구창은 그립(⠿)을 끌어 원하는 자리로 이동(악보를 스크롤해도 그 자리에 떠 있음). 줌·재생 바는 상단바에 고정이라 끌 필요가 없습니다.
- 편집 독은 위 경계선을 끌어 높이를, 선율 팔레트↔에디터 사이 경계를 끌어 너비를 조절.
- 사이드바는 ✕로 접고 "설정" 버튼으로 다시 엽니다.
- 버튼에 마우스를 올리면 설명 말풍선이 뜹니다(화면 가장자리에서도 잘리지 않게 위치 자동 보정).
- 각 입력창 머리의 **?** 버튼: 입력 방법·문법 안내. 첫 방문 때 선율 안내가 한 번 자동으로 열립니다.
- **단축키** 버튼(에디터 모드 편집 독 레일): 전체 단축키 안내. 직접 입력 모드에선 이 레일이 숨겨지므로 상단바 **도움말 → 단축키** 탭을 참고합니다.
- 직접 입력 모드의 기능바는 **기본이 왼쪽 세로 사이드**입니다(접기 옆 위치 전환 버튼으로 악보 위 가로 배치와 오감): 접기·되돌리기가 맨 위 한 줄로, 입력 버튼이 3:2로, 열린 도구창이 그 아래로 도킹되어 사이드바처럼 정리됩니다.
  - **폭 조절**: 도크와 악보 사이의 세로 알약 손잡이를 **끌면 열 폭이 조절**됩니다(최소 240px). **더블클릭하면 자동 폭**으로 복귀. 지정한 폭은 저장되어 유지됩니다.
  - 어떤 도구창을 켜도 열 폭이 같고, **시김새·장단·가사 기호 팔레트는 같은 격자**라 열 폭에 맞춰 한 줄 개수가 자연스럽게 줄바꿈됩니다(기본 폭에서 6개/줄). 율명 표는 폭이 모자라면 표 안에서 가로 스크롤.
  - 율명 팔레트를 크기 ±로 키워도 열 폭은 그대로 — 커진 표는 팔레트 안 가로 스크롤로 봅니다.
- **다크 모드**: 상단바 오른쪽 **달/해 아이콘 + 글자(다크/라이트)** 버튼으로 밝은 화면 ↔ 어두운 화면 전환(선택은 이 컴퓨터에 유지). UI만 어두워지고 **악보(종이)는 늘 흰색**이라 인쇄·PNG 출력은 그대로입니다. 다크에선 시그니처 색이 **파란색**으로 바뀌어 가시성이 높고, 검은 선으로 그려진 시김새·장단 팔레트 기호도 흰 선으로 반전되어 잘 보입니다(악보 위 기호는 검정 유지).

## 15. 단축키 모음

| 키 | 동작 |
|---|---|
| `⌘/Ctrl+Z` / `⌘/Ctrl+⇧+Z` | 실행 취소 / 다시 실행 |
| `⌘/Ctrl+Enter` | 각 추가(선율 에디터) |
| `Enter` | 각 나누기(선율 에디터, 커서 위치) |
| `Enter` / `Tab` | 다음 정간(직접 입력 카드) |
| 방향키 | 인접 정간 이동(직접 입력 카드) |
| `Esc` | 입력 카드 닫기 |
| `→` | 다음 정간으로 커서 이동(텍스트 에디터) |
| `1`~`0`(누른 채) | 시김새 골라 음 클릭해 붙이기 — **누르고 있는 동안만** 적용, 떼면 해제(직접 입력, 팔레트 배지 참고) |
| `Backspace`/`Delete` | 선택한 시김새 삭제(미세조정 모드) |

## 16. 알려진 제약 / 참고

- 임시 저장·자동 저장은 **그 컴퓨터의 그 브라우저 안에만** 남습니다. 다른 곳으로 옮기려면 "파일로 저장"을 쓰세요.
- 셀 서식(배경색·테두리)은 아직 선율 정간에만 적용됩니다(장단·가사 칸 미지원).
- 시김새·장단은 재생 소리에 반영되지 않습니다(선율만 재생).
- 옥타브 변형 한자는 ±2옥타브까지 지원하며, 변형자가 없는 2개 조합(하배이·중청대)은 옥타브 점으로 대체됩니다.
- `symbol_svgs/` 원본을 수정해도 앱은 `symbols-data.js`를 읽으므로 데이터 재생성이 필요합니다.
